Denovo RNASeq

We offer the following approach to analyse RNASeq data if the reference genome is not available at public resources like NCBI, ENSEMBL, JGI or others. This might be due to a variety of reasons including lack of a reference altogether amongst many. Normalization of Reads: Reads will be normalized using a standalone approach to ease the computational load and to correct for coverage of ultra-high/ultra-low kmers to assemble these transcripts.

Eukaryotic Genome Assembly

We perform Eukaryotic Genome Assemblies to generate the following information about the prospective organism. Genome Survey: Determine metrics like Genome size, heterozygosity and repetitive content of the genome. Genome Assembly: Create contiguous sequence segments using the sequenced reads representing the genome. Assembly will be validated using BUSCO, Transcript mapping (subject to availability) and Merqury to assess the genome completion. Genome Masking: Identify low complexity, mobile elements and simple repeats to mine features.

Exome Analyses

We offer the following approach to analyse Exome data. The primary requisite information is the kit that was used to prepare the library and the associated BED file so as to identify the targeted regions. Alignment of the reference genome: Depending on the kit and the associated BED, we will be aligning the reads to the appropriate version: Hg19, Hg38, Grch37, Grch38. Variation calling: For Tumor-Normal matched samples, somatic variation calling will be done.

Genome Survey

Genome Survey analyses at Ikigai entails extraction of the following information from the datasets using a kmer based approach. Since Genome Survey is done at 25-30X data (i.e 1Gb genome => 30Gb data), we can analyse the data in the below fashion. Genome size: Ballpark the size of the genome and mark it against existing literature. Heterozygosity of genome: Heterozygosity corelates with the presence of allelic variations, which might complicate genome assembly.

Prokaryotic Genome Assembly

We perform Prokaryotic Genome Assemblies to generate the following information about the prospective organism. Data QC and Contamination screen: Low quality sequences and bases are removed. Subsequently 10000 reads are picked randomly to screen and calculate the proportion of the organism of interest. Genome Assembly: Create contiguous sequence segments using the sequenced reads representing the genome. Please let us know if Plasmids are also present in the data.

Reference Aided DNASeq

We offer the following approach to analyse WGS data if the reference genome is readily available at public resources like NCBI, ENSEMBL, JGI or others. Alignment of the reference genome: Reads will be aligned to the reference genome to check the mappability, coverage and depth of the organism sequenced. Variation calling: For the samples, we will be performing variant calling to call SNVs, InDels and smaller complex variations like MNPs (<50bp).
Analyses

Reference Aided RNASeq

We offer the following approach to analyse RNASeq data if the reference genome is readily available at public resources like NCBI, ENSEMBL, JGI or others. We prefer having the genome from ENSEMBL due to the downstream integration with biomart. Alignment of Reads: Reads will be aligned with the genome using the splice-sites information if available depending on the organism (prokaryotes will generally not have). Novel transcript assembly: Novel transcripts will be assembled using the help of the genome and screened for their coding potential.
Analyses

smallRNA Analyses

We offer the following approach to analyse RNASeq data if the reference genome is readily available at public resources like NCBI, ENSEMBL, JGI or others. We prefer having the genome from ENSEMBL due to the downstream integration with biomart. QC and Read Trimming: Trim adapters and retain reads between 18-30bp. Alignment to other ncRNAs: ncRNA catalogue from which the smallRNA sequences (precursors or mature) have been removed are used as a bait to remove reads originating from such sequences.
