ANALYSES

Prokaryotic Genome Assembly

We perform Prokaryotic Genome Assemblies to generate the following information about the prospective organism.

  1. Data QC and Contamination screen: Low quality sequences and bases are removed. Subsequently 10000 reads are picked randomly to screen and calculate the proportion of the organism of interest.

  2. Genome Assembly: Create contiguous sequence segments using the sequenced reads representing the genome. Please let us know if Plasmids are also present in the data. Genome assembly will be validated using BUSCO at the lineage pertaining to the species.

  3. Genome Annotation: Identify and mine features coding for proteins or their regulatory elements. We will also identify CRISPR elements in the genome.

  4. Virulence Factor Identification: We use the databases: VFDB, ResFinder and NCBI AMRFinderPlus to identify the genes related to Virulence factors

  5. Phylogenetic and PanGenome: Minimizer based approach to generate Phylogenetic tree based on Jaccard or other metrics. PanGenome analyses to identify the presence of Core genes and generate a gene-based phylogenetic tree.

  6. Variant Identification: Variation Calling with respect to a reference or Genome-vs-Genome alignments. Variation will be annotated with respect to the reference.

  7. MLST: Screening against PubMLST database to identify the various schemas associated with the organism
