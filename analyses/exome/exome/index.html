ANALYSES

Exome Analyses

We offer the following approach to analyse Exome data. The primary requisite information is the kit that was used to prepare the library and the associated BED file so as to identify the targeted regions.

  1. Alignment of the reference genome: Depending on the kit and the associated BED, we will be aligning the reads to the appropriate version: Hg19, Hg38, Grch37, Grch38.

  2. Variation calling: For Tumor-Normal matched samples, somatic variation calling will be done. Germline variation calling will be done for an unmatched sample.

  3. Copy number identification: For unmatched samples, we will be using HMM based approach to identify if there are Copy Number Variations if any in the sample.

  4. Annotation of Variation: Variations will be annotated against a plethora of databases like OMIM, COSMIC, dbSNP, Sift and LoFT to assess the impact of the variants observed.
